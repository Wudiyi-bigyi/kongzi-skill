这是一个基于《论语》孔子语录，为各类人生，学习，处事问题提供儒家经典解读与回答。

## 效果预览

### 怎样才能静下心坚持学习

> **子曰:** "知之者不如好之者，好之者不如乐之者。"欲静心学习，当先培养兴趣，以学为乐，而非以学为苦。"学而时习之，不亦说乎？"学习贵在"时习"，每日精进，积少成多。勿急于求成，"无欲速，无见小利"——欲速则不达，见小利则大事不成。"学而不思则罔，思而不学则殆。"学思并重，方能入脑入心。心不静者，多因杂念纷扰。当以"吾日三省吾身"之法，时常自省，去除浮躁。"君子务本，本立而道生。"学习之根本，在于修身。内心修养深厚，自然能静下心来。如《诗》云："静女其姝，俟我于城隅。"心静而后能学，学而后能明，明而后能行。"三人行，必有我师焉。"以他人为师，以经典为师，以生活为师。谦虚好学，不耻下问，自然心生敬畏，杂念自消。总而言之，静心学习之法有三：一曰培养兴趣，以学为乐；二曰循序渐进，持之以恒；三曰修身养性，去除浮躁。三者兼备，自然能静下心来，学有所成。

### 朋友犯错我该怎么劝告他

> **子曰：** 子曰："忠告而善道之，不可则止，毋自辱焉。"劝告朋友，当以真诚之心，温和之道。《论语》有云："君子之交淡如水"，朋友相处，贵在真诚。若朋友犯错，汝当直言相告，然须讲究方式。"善道之"者，乃循循善诱，而非厉声指责。如吾之教弟子，或问、或喻、或引经据典，使其自悟。汝可曰："吾闻之，'过则勿惮改'，君子之过也，如日月之食焉：过也，人皆见之；更也，人皆仰之。"如此以理服人，以情动人。"不可则止"者，乃知进退也。若朋友不听，不可强求，更不可恶言相向。"毋自辱焉"——强劝不听，徒增嫌隙，非交友之道。再者，"与朋友交，言而有信"，汝之言，须出于真心，而非为显示己之长。"见不贤而内自省也"——朋友之过，亦可为汝之镜，反观自身，是否有类似之失。总而言之，劝告朋友之法有三：一曰真诚相告，不隐瞒；二曰循循善诱，不指责；三曰知进退，不强求。三者兼备，既尽朋友之责，又不伤朋友之情。

### 常常为还没有发生的事而焦虑不安

>**子曰：** 子曰："君子坦荡荡，小人长戚戚。"为未发生之事焦虑，乃内心不静之故也。"人无远虑，必有近忧"，适当思虑未来，本无可厚非。然过度焦虑，则如杞人忧天，徒增烦恼。《中庸》云："喜怒哀乐之未发，谓之中；发而皆中节，谓之和。"心有所忧，当使其适中，不可过也。"无欲速，无见小利"——汝之所忧，或因急于求成，或因患得患失。须知世事难料，尽人事而听天命可也。"不怨天，不尤人"，反求诸己，做好当下之事，便是对未来最好的准备。"吾日三省吾身"——时常自省，问自己："今日之事，我尽力乎？"若已尽力，则心安；若未尽力，则当勉励。心有所安，则焦虑自消。再者，"学而时习之，不亦说乎？"以学习充实内心，以修身养性，自然能"内省不疚，夫何忧何惧？"君子务本，本立而道生，内心修养深厚，自能从容面对未来之变。总而言之，化解焦虑之法有三：一曰做好当下，不预支未来之忧；二曰修身养性，以学养心；三曰顺其自然，尽人事听天命。如此，则"君子坦荡荡"，无忧无惧矣。
